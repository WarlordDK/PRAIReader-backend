# **PRAIReader**

AI-анализатор презентаций, использующий визуально-языковые модели (VLM), нейросетевые эмбеддинги и векторную базу данных Qdrant для глубокого анализа содержимого слайдов.

---

##  **Развёртывание и установка**

###  Настройка переменных окружения

Перед запуском необходимо создать файл `.env` в корне проекта и настроить параметры:

```
HUGGINGFACE_HUB_TOKEN=
QDRANT_URL=
QDRANT_API_KEY=
```

---

##   **Получение HUGGINGFACE_HUB_TOKEN**

### Шаги:

1. Перейдите на сайт: [https://huggingface.co/](https://huggingface.co/)
2. Авторизуйтесь в свой аккаунт (или зарегистрируйтесь).
3. Нажмите на аватар → **Settings**.
4. В левом меню выберите **Access Tokens**.
5. Нажмите кнопку **New token**.
6. Укажите:

   * **Name:** любое название
   * **Role:** обязательно **Write**
7. Нажмите **Create token**.
8. Скопируйте токен и вставьте его в `.env`:

```
HUGGINGFACE_HUB_TOKEN=hf_**********************************
```

---

## **Подключение Qdrant**

PRAIReader использует векторную базу данных **Qdrant** для хранения эмбеддингов текстов и последующего поиска по ним.

### Qdrant Cloud 

1. Перейти на [https://cloud.qdrant.io](https://cloud.qdrant.io)
2. Создать аккаунт.
3. Нажать **Create Cluster**.
4. Выбрать минимальный тариф (Free Tier доступен).
5. После создания:
   * Скопировать **Cluster URL**
   * Скопировать **API Key**
6. Добавить в `.env`:

```
QDRANT_URL=https://*****.*****.*****
QDRANT_API_KEY=**********************************
```

##  **Запуск проекта через Docker Compose**

Ориентировочный вес контейнера ~15 gb.

Запуск:

```
docker-compose up --build
```

После сборки приложение будет доступно по адресу:

```
http://localhost:8000
```
